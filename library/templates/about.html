Let’s back to this system, the first thing you see when you open the system is, home page which will list the post made by the user who have access. Then, when open book page, it will show you available shelves created to hold the books as per shelf category. `When you create shelves, please make sure to be specific. It will help you when you on categorizing the books. For example, `it is best creating python book shelf than programming book shelf. ` Then when you click the shelf, it will direct you to the list of books added in that shelf. Then the site has about and contact page for general information. There is also navigation to direct you to standard admin page. It will prevent you to write on address bar every time you want to access the admin page.
